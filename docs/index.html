ECOBIN

ECOBIN은 환경 보호를 위한 올바른 재활용 방법을 안내합니다. 자원을 올바르게 분리배출 하는 방법을 제공하고, 현재 내 위치 근처의 탄소중립포인트 사용처 정보를 제공하고 실시간으로 걸음 수에 비례한 탄소저감량을 표시함으로써 재활용 활동을 더욱 편리하고 효과적으로 만들고자 하는 목적을 가진 애플리케이션입니다.

오늘 뭐할까?

사용자의 위치와 취향을 기반으로 식당·카페·여가시설 등을 추천해 주는 웹 서비스입니다. Supabase와 FastAPI로 구축한 API 서버를 통해 장소 정보, 메뉴, 영업시간 등을 제공하며, 사용자는 찜·조회 기록을 기반으로 개인화된 추천을 받을 수 있습니다. React 기반 UI로 코스 생성, 예산·거리 필터링, 상세 정보 조회 등 다양한 기능을 제공합니다.
